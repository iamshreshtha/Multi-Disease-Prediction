# Multi Disease Prediction System using Machine Learning
Project Overview

# The Multi Disease Prediction System is an AI-powered healthcare application that predicts the likelihood of four major diseases using machine learning models:

🩸 Diabetes Prediction
❤️ Heart Disease Prediction
🧪 Chronic Kidney Disease Prediction
🧬 Liver Disease Prediction

### The system provides an interactive web interface built with Flask, allowing users to enter medical parameters and receive instant predictions.

### Features
Predicts Diabetes using clinical measurements.
Predicts Heart Disease using lifestyle and health indicators.
Predicts Chronic Kidney Disease (CKD) using laboratory test results.
Predicts Liver Disease using liver function parameters.
User-friendly tabbed interface with responsive design.
Real-time prediction through a Flask web application.
Easily deployable on platforms such as Render and Railway.

Installation

# Clone the repository:

git clone https://github.com/yourusername/MultiDiseasePrediction.git

Move into the project directory:

cd MultiDiseasePrediction

Install dependencies:

pip install -r requirements.txt
Running the Application

Start the Flask server:

python app.py

Open your browser and visit:

http://127.0.0.1:5000
Deployment

This project can be deployed using:

Render
Railway
PythonAnywhere
Docker

# For Render deployment:

Create a Procfile:

web: gunicorn app:app

Create a runtime.txt file:

python-3.12.4
